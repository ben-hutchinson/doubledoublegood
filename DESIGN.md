Brand Direction

Minimal / Trusted / Local

Visual Language
Clean whitespace
Strong typography
Monochrome base palette
Subtle accent only if needed
Real photography over decoration
Layout System
Max width containers
Generous vertical rhythm
Clear hierarchy
Predictable navigation
Typography
Bold headings
Readable body text
Strong contrast
Consistent scale
Components
Header

Clickable logo linked to home. Primary nav order: About, Find Us, Sell, Contact, Reviews. Social links shown in header. On mobile, use stacked visible links rather than a hamburger.

Footer

Address, phone, email, opening hours, copyright, mailing signup, repeated footer nav, Delivery & Returns link, Privacy Policy link.

Cards

Used for carousel items, reviews, info blocks.

Forms

Simple labels, clear validation, large tap targets.

Responsive Rules
Mobile first
Stack content on small screens
Expand to columns on larger screens
Keep nav simple on mobile
Imagery

Use placeholders now. Replace with authentic store imagery before launch.

Design Reference

The approved V1 visual direction is the minimalist mockup stored in the repo as design_mockup.png.

When implementation details are unclear, use design_mockup.png as the primary visual reference for spacing, hierarchy, typography feel, layout proportions, and overall tone.

If written guidance conflicts with the mockup, follow the mockup unless that would harm accessibility or responsiveness.

UX Tone

Calm, credible, welcoming, easy to use.
